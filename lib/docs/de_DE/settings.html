Server und Browser

  • Sprache
    Diese Option stellt die für die Oberfläche und die Blöcke verwendete Sprache ein. Nach der Änderung ist ein erneutes Laden des Arbeitsfläche erforderlich. Achtung! Dar aktuelle Sketch sollte vorher gespeichert werden.

  • Starte abbozza! sofort
    Ist diese Option aktiviert, wird abbozza! direkt mit dem Start der Arduino Enticklungsumgebung gestartet.

  • Den Browser beim Start öffnen
    Ist diese Option aktiviert, wird beim Start der Arduino Entwicklungsumgebung auch der eingestellte Browser gestartet und die Arbeitsfläche geladen.

  • Pfad zum Browser
    Hier wird der Pfad des verwendeten Browsers eingestellt.

  • Server Port
    Der abbozza!-Web-Server ist unter diesem Port erreichbar. Im Zweifelsfall sollte der Standardport 54242 eingestellt werden.

  • Prüfe bei Start auf aktuelle Version
    Ist diese Option aktiviert, prüft abbozza! bei jedem Start, ob eine neuere Version verfügbar ist. Wenn ja kann Sie heruntergeladen und installiert werden. Achtung! Danach ist ein Neustart der Arduino Entwicklungsumgebung notwendig.

  • URL für aktuelle Version
    Unter dieser URL wird nach der aktuellen abbozza!-Version gesucht. Die Standardeinstellung ist inf-didaktik.rz.uos.de/abbozza/current/

  • Suche nach Update
    Diese Schaltfläche bewirkt, das nach einer aktueleren Version von abbozza! gesucht wird.
